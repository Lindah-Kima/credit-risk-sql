# Business Questions

This document tracks the business questions explored throughout the project.

Each question represents a real analytical problem that can be answered using SQL.

---

# Customer Analysis

* How many customer records exist in the lending system?
* Does every customer have a unique identifier?
* Which customer attributes contain missing values?
* Do any text fields contain blank values instead of meaningful information?
* How many new customers joined over time?
* Which customer segments generate the most loan applications?
* How are customers distributed by gender?
* How are customers distributed by education level?
* How are customers distributed by employment status?
* How are customers distributed across counties?
* How are customers distributed by KYC status?
---

# Loan Applications

* How many loan applications have been submitted?
* What is the application approval rate?
* Which loan products receive the most applications?
* What are the most common reasons for application rejection?

---

# Loans

* How many active loans exist?
* What is the total value of the loan portfolio?
* Which loan products generate the highest disbursement amounts?
* Which branches issue the most loans?

---

# Repayments

* What percentage of scheduled repayments have been completed?
* Which customers consistently repay on time?
* Which loans have overdue repayments?

---

# Credit Risk

* Which customers have the highest probability of default?
* Which factors are associated with higher credit risk?
* How has portfolio quality changed over time?

# Collections

* Which loans have entered collections?
* Which collection agents recover the most overdue balances?
* What is the recovery rate for delinquent loans?
